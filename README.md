# Digiline Official Website

The Digiline Official Website is a modern, responsive corporate platform built to showcase Digiline’s telecommunications solutions and services, including SMS, USSD, Voice, Airtime, and API integrations.

The website presents company solutions, real-world use cases, career opportunities, support resources, and contact channels through a clean, enterprise-grade user interface.

---

## Tech Stack

- Next.js
- Tailwind CSS
- React Icons
- Next.js Image Optimization

---

## Key Features

- Service & Solutions Pages (OTP, 2FA, USSD, SMS APIs)
- Career Page with Job Listings & Details
- Support Center with FAQ Accordion
- Contact Page with Inquiry Form
- Fully Responsive Design

---

## Getting Started

```bash
npm install
npm run dev
```

Visit: `http://localhost:3000`
